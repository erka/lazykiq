# lazykiq

<p align="center"><img width="512" src="./doc/assets/lazykiq.png" alt="lazykiq logo" /></p>

A terminal UI for Sidekiq.

## Usage

```bash
go run ./cmd/lazykiq
```

## Development

We use [`mise`](https://mise.jdx.dev/) for development. Run all CI tasks with:

```bash
mise run ci
```

To update all dependencies:

```bash
mise run deps
```

## Keys

- `1-6` - switch views
- `q` - quit
